# Jarvis Desktop 주요 기능 명세서 (v1.3.5)

Jarvis Desktop은 macOS 시스템 제어, 웹 자동화, 그리고 다양한 외부 서비스를 통합하여 사용자의 의도를 자율적으로 수행하는 **AI 에이전트 데스크톱 어플리케이션**입니다.

---

## 1. 자율형 AI 에이전트 엔진
- **통합 ReAct 루프**: 사용자의 자연어 명령을 분석하고, 목표 달성을 위한 일련의 작업을 스스로 계획하고 실행합니다.
- **멀티 프로토콜 지원**: Gemini, Claude, OpenAI 등 최신 LLM과 로컬 Ollama 모델을 지원합니다.
- **비공식 Web-AI 연동**: ChatGPT 웹 세션을 활용하여 API 비용을 절감하면서도 강력한 추론 능력을 제공합니다.
- **장기 기억 (Long-term Memory)**: 사용자의 신원, 선호도, 과거 대화 맥락을 기억하여 개인화된 응답을 제공합니다.

## 2. macOS 시스템 자동화 (OS Automation)
- **네이티브 앱 제어**: 앱 실행(Launch), 종료(Quit), 활성화(Focus), 창 전환 등을 수행합니다.
- **사용자 입력 모사**: 키보드 입력(Typing), 단축키 실행, 마우스 클릭(Coordinate/UI element 기반)을 지원합니다.
- **화면 인식 (Screen Perception)**: macOS Vision 프레임워크 기반의 OCR을 통해 화면 내 텍스트와 창 구조를 파악합니다.
- **Finder 및 파일 시스템**: 파일 찾기, 폴더 이동, 파일 정보 조회 등의 작업을 자동화합니다.
- **시스템 알림 모니터링**: 앱 알림을 실시간으로 감지하고 대응할 수 있습니다.

## 3. 웹 자동화 (Web Automation)
- **자율 브라우징**: Playwright/Puppeteer를 통해 웹사이트를 탐색하고 클릭, 입력, 스크롤 등의 작업을 수행합니다.
- **DOM 인식**: 웹 페이지의 구조를 분석하여 타겟 요소를 정확히 찾아냅니다.
- **멀티 탭 관리**: 여러 웹 페이지를 동시에 다루며 정보를 수집하거나 작업을 수행합니다.

## 4. 커뮤니케이션 및 외부 서비스 통합
- **Discord & Slack**: 활성 대화방 모니터링, 메시지 읽기 및 답장 자동화를 지원합니다.
- **GitHub**: 레포지토리 관리, Issue/PR 확인, GitHub Actions 워크플로우 실행 및 모니터링이 가능합니다.
- **OBS Studio**: 스트리밍 및 녹화 제어, 씬 전환 등을 자동화할 수 있습니다.

## 5. 멀티미디어 및 사용자 인터페이스
- **음성 인터페이스 (STT/TTS)**: 음성 명령 인식 및 고품질 음성 합성을 통한 대화가 가능합니다.
- **프리미엄 사이드바 UI**: Next.js 기반의 ChatGPT 스타일 현대적 디자인을 적용하여 채팅 내역과 설정을 관리합니다.
- **개인정보 보호 (PII Manager)**: 비밀번호, API 키 등 민감한 정보를 안전하게 저장하고 에이전트가 필요할 때만 호출하여 사용합니다.

## 6. 확장성 및 유지보수
- **플러그인 시스템**: Connector(앱 연동), Skill(작업 힌트), Webhook을 통해 기능을 무한히 확장할 수 있습니다.
- **자동 업데이트**: GitHub Releases와 연동되어 앱 내에서 최신 버전을 간편하게 확인하고 업데이트합니다.
- **패키징 지원**: Electron을 사용하여 macOS용 `.dmg` 설치 파일 빌드 및 배포를 지원합니다.
