무료 다운로드

홍보 페이지에서 기능을 보고, 전용 다운로드 페이지에서 바로 설치를 시작할 수 있습니다.

Natural Intent

말한 목적을 먼저 이해하고 실행 흐름으로 잇습니다.

앱 이름과 사이트 이름, 작업 의도를 먼저 파악하는 방향으로 UX를 설계했습니다. 답변보다 실행이 앞서는 데스크톱 경험을 목표로 합니다.

Local Control

브라우저와 컴퓨터를 직접 다루는 데스크톱 AI.

음성 입력, 앱 실행, 브라우저 제어, 파일 작업까지 한 스레드 안에서 이어지는 구조를 지향합니다.

Guided Install

설치 전에도 제품의 성격과 권한 흐름을 분명하게 보여 줍니다.

홍보는 더 매력적으로, 다운로드는 더 간결하게, 동의 과정은 더 명확하게 분리해 사용자가 헷갈리지 않도록 정리했습니다.

Platforms

운영체제를 고른 뒤, 바로 다운로드 페이지로 이어집니다.

메인 페이지에서는 제품을 소개하고, 다운로드 페이지에서는 macOS·Windows·Linux 가운데 원하는 환경을 골라 약관 확인 후 설치를 시작합니다.
